## 3 · Grammar concept: particle consolidation

You now know the core particles of Japanese. Here's a recap table — memorize it cold. Full index at [grammar/particles-index.md](./grammar/particles-index.md).

| Particle | Role |
|---|---|
| は (wa) | Topic marker ("as for X…") |
| が (ga) | Subject marker / new info |
| を (wo, "o") | Direct object |
| の (no) | Possessive / modifier |
| に (ni) | Destination, specific time, existence-place, recipient |
| で (de) | Location of action, means |
| と (to) | "And" (exhaustive), "with" (companion) |
| も (mo) | "Also / too" |
| か (ka) | Question marker (sentence end) |

Still to come in phase 1: から, まで.

## 4 · Reading practice

Read aloud and translate:

1. 私は電車で駅に行きます。
2. この本は田中さんの本です。
3. 水を飲みます。
4. 車の下に猫がいます。
5. 彼も日本語がわかります。

<details><summary>Translations</summary>

1. "I go to the station by train."
2. "This book is Tanaka-san's book."
3. "I drink water."
4. "There's a cat under the car."
5. "He also understands Japanese."

</details>

## 5 · Speaking practice

Construct five sentences using each of these particles at least once: は, を, に, で, の, と, も. You can chain them.

Example run:
- 私は朝、家でコーヒーを飲みます。
- 明日、友達と駅に行きます。
- 田中さんも車を買いました。
- 机の上に本があります。
- 私も日本語がわかります。

## 6 · Quiz — broad particle review

Fill in the blank with the best particle:

1. 私 ___ 学生です。
2. パン ___ 食べます。
3. 駅 ___ 行きます。
4. 家 ___ 食べます。(eating at home)
5. 田中さん ___ 本 (Tanaka-san's book)
6. 友達 ___ 行きます。("with")
7. 私 ___ 学生です。("also")
8. これは何です ___ ？

<details><summary>Answers</summary>

1. は  2. を  3. に  4. で  5. の  6. と  7. も  8. か

</details>

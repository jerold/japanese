# Day 4 — Katakana Intro & Pronouns

**Target time:** 15–20 minutes.

## 1 · Kana onboarding

Katakana time. Same 46 sounds as hiragana, different shapes. Many katakana are stripped-down versions of the hiragana you already know:
- あ → ア, い → イ, う → ウ, え → エ, お → オ
- か → カ, き → キ, く → ク, け → ケ, こ → コ
- さ → サ, し → シ, す → ス, せ → セ, そ → ソ

Read the three rows above aloud. Notice how カ looks like a stripped-down 力 and how ケ is almost the kanji 介. These visual mnemonics stick.

Full chart: [vocab/kana.md#katakana](./vocab/kana.md#katakana).

## 2 · New vocabulary (4)

Today you meet the rest of the basic pronouns. Note: **Japanese drops pronouns whenever context is clear** — don't overuse these.

| Kanji (kana) | Romaji | English |
|---|---|---|
| あなた | anata | you |
| 彼 (かれ) | kare | he / boyfriend |
| 彼女 (かのじょ) | kanojo | she / girlfriend |
| 私たち (わたしたち) | watashitachi | we / us |

Detail: [vocab/pronouns-and-demonstratives.md](./vocab/pronouns-and-demonstratives.md).

⚠️ **あなた warning**: it often sounds distant or presumptuous. Prefer calling people by **name + さん** (e.g., 田中さん). See [culture/name-suffixes-san-kun-chan.md](./culture/name-suffixes-san-kun-chan.md).

## 4 · Grammar concept: the topic particle は

You've been using は in 私は. Today let's name what it does.

**は (pronounced *wa*) marks the topic** — "as for X…"

```
[Topic] は [comment] です。
```

- 私**は**学生です。 — "As for me, I'm a student."
- 彼**は**先生です。 — "He is a teacher."
- 彼女**は**田中さんです。 — "She is Tanaka-san."

The topic is whatever the sentence is *about*. Once it's established, you can drop it in follow-up sentences.

Full page: [grammar/particle-wa-vs-ga.md](./grammar/particle-wa-vs-ga.md).

## 5 · Reading practice

Read aloud, then translate:

1. 私は田中です。
2. 彼は先生です。
3. 彼女はアメリカ人です。 (amerika-jin = American person)
4. 私たちは友達です。 (tomodachi = friend)

<details><summary>Translations</summary>

1. "I am Tanaka."
2. "He is a teacher."
3. "She is American."
4. "We are friends."

</details>

## 6 · Speaking practice

Substitute in real details. Say aloud:
- 私は [your name] です。
- 私は [your country/city] 人です。
  - アメリカ人 (amerika-jin), イギリス人 (igirisu-jin, British), 日本人 (nihon-jin, Japanese)…

## 7 · Quiz

1. Translate: 彼は田中さんです。
2. What particle marks the topic?
3. Why is ~あなた~ risky to use with new people?
4. Complete the katakana line: ア, イ, ウ, __, オ.

<details><summary>Answers</summary>

1. "He is Tanaka-san."
2. は (pronounced wa)
3. It can sound distant, presumptuous, or overly intimate. Use the person's name + さん instead.
4. エ (e)

</details>

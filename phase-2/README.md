# Phase 2 — Scaffold (Stubs)

This folder holds the scaffold for phase 2. Files below are **stubs** — intentionally empty outlines to be filled in once phase 1 is complete.

See [../phase-2-outline.md](../phase-2-outline.md) for the full plan (themes, word counts, grammar points, suggested 40–50 day arc).

## Why stubs

Keeping the stubs visible does three things:
1. Makes the project structure complete so you can see the whole arc at a glance.
2. Lets you jot notes into a relevant file as they occur to you during phase 1 — no need to scaffold mid-flow.
3. Keeps the reference-folder convention consistent: every content area has a file.

## What goes in each file

When you fill each stub, keep the same format as the phase-1 reference files:
- A short intro paragraph.
- Sections with **stable heading anchors** that daily lessons can link to (`## word-anchor` style — kebab-case).
- Kanji + kana + English for every word entry (drop romaji in phase 2).
- A "See also" footer linking to related reference files.

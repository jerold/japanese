# か — The Question Particle

To turn a Japanese sentence into a question, put **か** at the end. That's it — no word-order change needed, no rising intonation required (though rising intonation is fine too).

## Basic pattern

```
[Statement] + か？
```

- これは本です。→ これは本ですか？ = "Is this a book?"
- 日本語を話します。→ 日本語を話しますか？ = "Do you speak Japanese?"
- 元気です。→ 元気ですか？ = "Are you well?"

## With question words

When you already have a question word (何, 誰, どこ…), add か at the end anyway — they work together:
- 何ですか？ (nan desu ka?) = "What is it?"
- どこですか？ (doko desu ka?) = "Where is it?"
- 誰が来ましたか？ = "Who came?"
- いつ行きますか？ = "When will you go?"

## Answering yes/no questions

- **はい** (hai) = yes / affirmative
- **いいえ** (iie) = no

Common short answers:
- はい、そうです。(hai, sou desu.) = "Yes, that's right."
- いいえ、違います。(iie, chigaimasu.) = "No, that's different / wrong."

## Casual question form

In casual speech you drop か and rely on intonation:
- 元気？ (genki?) = "You good?"

But for phase 1, use か.

## A gentler question feel: ～ですか？ vs ～ですよね？

Adding **ね** to the end softens a statement into something like "right?":
- 寒いですね。(samui desu ne.) = "It's cold, isn't it?"
- 田中さんですよね？ = "You're Tanaka-san, right?"

ね is a confirmation particle — you're seeking agreement, not information. (Full coverage in phase 2.)
